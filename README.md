# timothi_skill

Portable skills for Codex and Claude Code.

## diary-zt

`diary-zt/` saves visible conversations as Markdown diaries:

- `/diary-zt [标题]` creates a new full snapshot.
- `/diary-zt-a [已有文件名.md]` appends only the new content from the current session.

See [diary-zt/references/testing.md](diary-zt/references/testing.md) for installation and cross-device checks.
